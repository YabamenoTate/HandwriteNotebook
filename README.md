# 究極のレポートその2 Ultimate "Handwritten" Note Generator
令和の時代、我々は高度な計算機をポケットに入れ、瞬時に世界中の情報にアクセスできる世界に生きています。それにもかかわらず、いまだに「手書きレポート」や「手書きの課題」を要求する化石のようなタスクが教育機関には蔓延っています。「手書きの方が心がこもっている」「コピペ防止になる」などと指導側は供述していますが、実際に評価されているのは学力や論理的思考力ではなく、**純粋な手首の耐久力と苦行への耐性**です。腱鞘炎になるために高い学費を払っているわけではありません。

そんな理不尽な世界へのささやかな、しかし圧倒的な技術力を伴う抵抗として生まれたのがこのプログラムです。ぜひ適当に活用してください。

## 概要

**「確かにキーボードで打ち込んだ。だが、紙にあるこれは間違いなく手書きだ」**

本プログラムは、LuaLaTeXの強力なスクリプト処理機能を最大限に活用し、入力されたデジタルテキストを**「極めて人間臭い手書きノート」**へと変貌させるLaTeXテンプレートです。単に手書き風フォントを適用しただけの見破られやすいレポートとは次元が違います。

## このプログラムの凄さ（機能一覧）

Luaスクリプトがテキストを1文字単位でパースし、LaTeXコマンドや数式モードを賢く回避しながら、通常の文字列に対して以下のランダム化処理をリアルタイムで実行します。

* **複数フォントのランダム混植:** `ShigotoMemogaki`と`ERIJI`を1文字ごとにランダムで切り替え、同一人物の筆跡に生じる「どうしようもないブレ」を完全再現。（事前にフォントのインストールが必要です。）
* **フォントサイズのランダム化:** 12pt〜15ptの間で1文字ずつ大きさが変動。
* **縦横比（スケール）のランダム化:** 横幅が90%〜120%でランダムに伸縮し、文字の潰れや間延びを表現。
* **角度（回転）のランダム化:** 文字がベースラインから不規則に傾き、急いで書いたときの斜体感を演出。
* **文字間隔・上下位置のランダム化:** 文字と文字の間が詰まったり、不自然に空いたり、上下に踊ったりするガタツキを精密にシミュレート。
* **筆圧（インクの濃淡）のランダム化:** 文字ごとに黒〜グレーの色深度をランダムに変え、ボールペンのインクだまりや筆圧のムラを表現。
* **ノートレイアウトの完全再現:** `geometry`パッケージを用いて、市販のキャンパスノートの余白（上下、左右のミリ単位の寸法）を忠実に再現。
* **段落・箇条書きの最適化:** 箇条書きの不自然な余白を消去し、セクション見出しを「＜見出し＞」のようなノートテイストに自動置換。

## 必須環境と依存フォント

* **コンパイラ:** LuaLaTeX 必須
* **必要フォント:**
	* `ShigotoMemogaki` (しごとメモ書き)
	* `ERIJI` (えり字)

※必要に応じて手書きフォントを変えれば良いため、これらはあくまで一例でしかありません。

## 使い方
1. LuaLaTeXが実行可能な環境を用意します。
2. 上記のフォントをシステムにインストールするか、プロジェクトディレクトリに配置します。
3. このプリアンブルをドキュメントに組み込み、`\randomspacing{...}` コマンドを用いて本文を記述します。
4. LuaLaTeXでコンパイルし、出力されたPDFを印刷します。
5. 印刷した紙をほんの少しだけ曲げて、手汗っぽい水をかけたらボールペンを添えて、その写真を提出します。
5. 涼しい顔で「手書きしてきました」と言います。
